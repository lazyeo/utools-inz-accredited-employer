# 新西兰认证雇主查询 uTools 插件

一个基于 uTools 平台的插件，用于快速查询新西兰移民局 (INZ) 的认证雇主列表。

## 功能特点

- **原生集成**：完美适配 uTools 的搜索框交互。
- **实时查询**：直接连接 INZ 官网 API，数据每日更新。
- **一键复制**：点击搜索结果即可自动复制 NZBN 编号。
- **视觉适配**：支持系统暗黑模式。
- **分页加载**：支持查询大量结果时的分页加载。

## 开发与安装

### 1. 准备环境
确保你已经安装了 `Node.js`。

### 2. 安装依赖
```bash
npm install
```

### 3. 开发模式
```bash
npm run dev
```
默认运行在 `http://localhost:5173`。

### 4. 在 uTools 中预览
1. 打开 uTools 搜索框，输入 `开发者工具`。
2. 点击 `新建项目`，选择项目根目录下的 `public/plugin.json`。
3. 唤起 uTools，输入 `认证雇主` 即可开始查询。

## 功能指令
- `认证雇主`
- `AEWV`
- `Accredited Employer`

## 技术栈
- React 19
- Vite 6
- uTools API
